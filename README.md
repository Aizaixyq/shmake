# shmake build system
A flexible build tool built using a shell is available for linux 

## Function
1. Precompile
2. Multithreading
3. Incremental compilation

## Install
    $ git clone https://github.com/Aizaixyq/shmake.git
    $ cd shmake
    $ sh install.sh

## Build
    $ mkdir build 
    $ cd build
    $ shmake ..
